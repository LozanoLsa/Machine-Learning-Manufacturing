PROJECT: 03_KNN_SUP
TITLE: Data Sources & Feature Engineering

--------------------------------------------------
OVERVIEW:
This dataset contains 1,847 motor operating records from a condition monitoring system. The ML task is binary classification to detect incipient failure states (0 = normal operation, 1 = incipient failure), with a 29.0% incipient failure rate. Source: simulated sensor data modeling two distinct motor operating regimes based on ISO 10816 alarm thresholds.

--------------------------------------------------
REAL-WORLD DATA SOURCES:

Feature                   | Source System              | Instrument
--------------------------|----------------------------|--------------------------
vib_rms_mms               | SCADA / Vibration Monitor  | Accelerometer — RMS amplitude at motor housing
vib_peak_to_peak_mms      | SCADA / Vibration Monitor  | Accelerometer — peak-to-peak amplitude
bearing_temp_c            | DCS / Thermal Monitor      | Thermocouple or PT100 at bearing housing
motor_current_a           | PLC / Electrical Monitor   | Current transducer — motor phase current
dominant_freq_hz          | SCADA / Frequency Analyzer | FFT spectrum analyzer — dominant vibration frequency
load_pct                  | Machine Load Monitor       | Load cell or PLC set-point register — % of rated capacity
incipient_failure         | Maintenance / Diagnostic Log | Condition assessment record per inspection cycle

--------------------------------------------------
VARIABLE DESCRIPTIONS:

vib_rms_mms            - RMS vibration amplitude (mm/s); normal ~2.2, failure ~4.3. Primary alarm signal reflecting ISO 10816 alarm thresholds.
vib_peak_to_peak_mms   - Peak-to-peak vibration (mm/s); normal ~5.0, failure ~9.0. Captures impact events and mechanical looseness.
bearing_temp_c         - Bearing temperature (°C); normal ~60°C, failure ~82°C. Indicates friction buildup and lubrication degradation.
motor_current_a        - Motor current draw (A); normal ~28A, failure ~35A. Reflects mechanical resistance increase.
dominant_freq_hz       - Dominant frequency (Hz); normal ~29.5 Hz, failure ~24.5 Hz. Misalignment signature in vibration or electrical signal.
load_pct               - Operating load as % of rated capacity; normal ~70%, failure ~85%. Operating context that modulates thermal signals.
incipient_failure      - Binary classification target: 0 = normal operation, 1 = incipient failure state.

--------------------------------------------------
FEATURE ENGINEERING:

StandardScaler applied to all six features — mandatory for KNN because distance calculations require features on comparable scales. All preprocessing inside a sklearn Pipeline. GridSearchCV over k x weights x distance metric; best: k=7, weights='distance', p=1 (Manhattan). Train/Test split: 80/20, random_state=42.

--------------------------------------------------
BUSINESS CONTEXT:

Used by maintenance engineers in industrial operations monitoring electric motors on production lines, conveyor systems, HVAC, and pumps. The model enables the shift from reactive to predictive maintenance by detecting incipient failure states from sensor patterns measurable weeks before a motor trips a protection relay.

--------------------------------------------------
GOAL:

Classify motor operating state as normal or incipient failure to enable predictive maintenance scheduling, reduce unplanned downtime, and improve MTBF. Model achieved Accuracy 88.3%, AUC-ROC 0.940, Recall (Failure) 71.4%.
